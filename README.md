## **WALLTOPIA - MODERN WALLPAPER PLATFORM**

### **Overview**

- Walltopia is a modern and responsive wallpaper platform designed to provide users with high-quality wallpapers for both desktop and mobile devices.
The project focuses on performance, clean design, and an intuitive user experience.

- It is built using a lightweight and efficient structure with JSON-based data management and Bootstrap-powered UI design.

### **🚀 Features**

- 🎨 High-resolution wallpaper collection
- 🔍 Search and category filtering system
- 📱 Fully responsive design (mobile-first)
- ⚡ Fast loading with optimized assets
- 🧭 Clean and user-friendly interface

### **🛠️ Tech Stack**

- HTML5
- CSS3
- JavaScript
- Bootstrap
- JSON (for data handling)
